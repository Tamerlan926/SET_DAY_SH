Я разработал Telegram-бота который помогает отслеживать ежедневное настроение, количество часов работы/учебы и сна. Моя цель была создать простой и удобный инструмент для самоанализа, который показывает, как образ жизни влияет на самочувствие

Интерфейс бота:
Я настроил удобное меню с кнопками чтобы пользователю не пришлось запоминать команды
Реализовал пошаговый опрос: сначала настроение смайлики потом часы работы и сна затем комментарий
Добавил кнопку Помощь, которая объясняет как пользоваться ботом

База данных:
Я выбрал SQLite, так как это легкий и надежный вариант для локального хранения данных
Создал таблицу где хранятся: дата, настроение set, часы работы, часы сна и комментарий

График:
С помощью библиотеки Pandas я написал функции которые считают среднее настроение сон и работу за неделю или месяц.
Используя Matplotlib я научил бота строить красивые графики динамики настроения и отправлять их пользователю в виде картинки
Я добавил блок Инсайты: бот сам анализирует данные и пишет например: Когда ты спишь больше 8 часов, настроение выше

Стабильность:
Я протестировал бота на ошибки ввода например, если вместо числа написать текст и сделал так чтобы бот вежливо просил повторить ввод.
Настроил работу через screen на сервере, чтобы бот работал круглосуточно и не выключался при разрыве соединения.
Структура моих файлов
bot.py — основной код
db.py — мои функции для записи и чтения данных из базы SQLite
stats.py — код для анализа статистики и рисования графиков
schema.sql — схема моей базы данных
requirements.txt — список библиотек которые я установил pyTelegramBotAPI, pandas, matplotlib
Назвал колонку настроения set в кавычках, потому что без кавычек SQL ругается потому что это зарезервированное слово

Так как у меня есть свой сервер я покажу 2 способа работы моего бота это локально и так скаазать глобальный

Запуск кода локально:
Сначало устанавливаем зависимости тут нам и нужна папка requirements.txt пишем в терминал - pip install -r requirements.txt
Вставляем свой токен в гланый код в bot.py
И запускаем бота пишем в терминал - python3 bot.py

Запуск кода глобально:
Сначала я захожу в терминал и подключаюсь к серверу командой:
ssh root@ip
Потом я создаю папку для проекта:
mkdir Set_day
Захожу в созданную папку
cd Set_day
Затем через редактор nano я создаю файлы например пишу
nano bot.py
Вставляю туда свой код сохраняю сочетанием клавиш Ctrl + O подтверждаю Enter и выхожу через Ctrl + X. Так же я создаю остальные файлы db.py, stats.py, schema.sql, requirements.txt
Перед установкой зависимостей я создаю виртуальное окружение чтобы всё работало корректно:
python3 -m venv venv
И активирую его:
source venv/bin/activate
Потом устанавливаю все необходимые библиотеки
pip install -r req.txt
Для проверки я делаю тестовый запуск чтобы убедиться что всё работает
python3 bot.py
Если ошибок нет и бот отвечает я останавливаю его Ctrl + C и настраиваю работу 24/7
Сначала устанавливаю утилиту screen
apt update
apt install screen -y
Запускаю бота внутри 
screen -S Set_day
Не забываю снова активировать окружение и запускаю бота
source venv/bin/activate
python3 bot.py
Чтобы выйти из экрана оставив бота работать в фоне я нажимаю комбинацию клавиш Ctrl + A а затем D
